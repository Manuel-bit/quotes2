## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Installation

open terminal and run  the command git clone https://github.com/Manuel-bit/quotes.git
navigate to the folder quotes and run the command ng serve

## Description

The project helps users to share quotes, a user can add quotes and vote for the best quotes.
